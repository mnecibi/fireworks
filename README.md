# Fireworks using P5.js 
Fireworks using P5.js ([inspired by the coding train youtube video|https://www.youtube.com/watch?v=CKeyIbT3vXI])

![Fireworks](fireworks.gif)

## First use
Install necessary node modules
```shell
npm install
```

## Usage
Build project
```shell
npm run build
```


Local server
```shell
live-server
```
